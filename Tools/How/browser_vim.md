# keymap

| key | descr |
|:---: | :---: |
| `j,k` | scroll up, down |
| `/` | search |
| `gi` | go to first input box |
| `gg,G` | page top/bottom |
| `zi,zo` | zoom page in/out |
| `H, L` | page history back/forword |
| `m<*>, ;<*>, '<*>` | mark: `m<*> firfox used, ;<*> chrome used |
| `o` | open |
| `gp` | toggle pin/unpin |
